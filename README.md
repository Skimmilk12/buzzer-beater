# 🏀 BUZZER BEATER

One-button rooftop basketball. Hold to charge, release to shoot, catch the teal band for a guaranteed swish. The shot clock is always ticking — and if the ball is in the air when the horn sounds, bank it for the **BUZZER BEATER** and play on in overtime.

## Features

- **Daily Challenge** — everyone in the world gets the same hoops, wind, and quarters each day. Share your score card and talk trash.
- **Quarters change the rules** — wind, double rims, sprinting meters, sniper bands, flickering lights, and CRUNCH TIME stacks them.
- **Twin hoops** — a safe 2 and a deep 3 up at once; your release power picks the shot.
- **Heat checks, streak multipliers, buzzer-beater overtimes**, six unlockable ball skins, synthesized lo-fi beat.

Runs anywhere as a single HTML file — installable as an app (PWA), works offline.

## Tech

Vanilla JavaScript, canvas, WebAudio (all sounds + music synthesized — zero asset files). No frameworks, no build step, one file.
